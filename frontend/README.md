# LOL Skin Collector - Frontend

This is a small Vue 3 + Vite frontend that currently exposes a single **Login** page for the LOL Skin Collector project.

## Getting started

From the `frontend` folder:

```bash
npm install
npm run dev
```

Then open the URL printed in the terminal (typically `http://localhost:5173`) to see the login page.

## Structure

- `index.html` – entry HTML file.
- `src/main.js` – bootstraps the Vue app.
- `src/App.vue` – root component, currently renders the login page.
- `src/components/LoginPage.vue` – main login UI.
- `src/assets/main.css` – global styling for the login page.
